# 🏡 Home Loan Prediction using Machine Learning

## 📌 Overview
This project predicts whether a customer will be eligible for a home loan using Machine Learning.

## 🚀 Features
- Loan eligibility prediction
- Clean UI using Flask + Bootstrap
- Data visualization dashboard
- Multiple ML algorithms used

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- HTML, CSS, Bootstrap

## 📊 Model Used
- Logistic Regression
- SMOTE for class balancing

## ▶️ How to Run
```bash
python app.py
```
